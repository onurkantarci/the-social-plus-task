# Social plus task


## Technologies used:
- Typescript, Vite, React, Tailwind, ShadCN (as UI Library), Zustand (State management)

## Running the Project

Follow these steps to run the project:

1. Install required packages:

   ```bash
   npm install
   ```

2. Run the React app using:

   ```bash
   npm run dev
   ```
